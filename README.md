# Sober Tracker Reader

A fancy PWA to track your sober time, money saved, and provide daily motivation.

## Features
- Animated counters for sober time & money saved
- Random motivational messages
- Glassmorphism & gradient UI
- Fully PWA-ready (offline & installable)
- Persistent data using localStorage

## Installation
1. Clone this repo or download files.
2. Place `icon-192.png` and `icon-512.png` in the root.
3. Open `index.html` in your browser.
4. On iOS Safari or Android Chrome, **Add to Home Screen** for PWA functionality.

## Usage
1. Enter the last time you used drugs.
2. Enter average daily cost.
3. Click **Save / Update**.
4. Watch sober time and money saved update live.
5. See rotating motivational messages.
6. Reset to start over.

## Hosting
- Can be hosted on any static web server (GitHub Pages, Netlify, Vercel, etc.)
- HTTPS recommended for PWA install.
